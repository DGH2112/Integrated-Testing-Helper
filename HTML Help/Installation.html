Installation

The Integrated Testing Helper is design to be loaded as a DLL expert. You can use GExperts (http://www.gexperts.org/) Experts Manager to do this or follow the instructions below:

  1. From the start menu select Run;
  2. Enter regedit and press >Enter>;
  3. Browse to the key HKCU\Software\Borland\BDS\4.0\;
  4. If there is no key called "Experts" then you will need to create one;
  5. Under "Experts" create a new string as follows:
    • Key = ITHelper2006;
    • Value = Drive:\Path\ITHelper2006.DLL.
The above example refers to RAD Studio 2006. For other version you may need to substitute Borland for either Codegear or Embarcadero and change the version number accordingly.

On reloading the IDE the IDE Experts should appear in the IDE.
